# 🕵️ Web Scraper with BeautifulSoup - Wikipedia Edition

🚀 **Scrape Wikipedia pages effortlessly using Python & BeautifulSoup!**

## 🌟 Features
- 📄 Grab **text, tables, and links** from Wikipedia.
- 🏎️ Fast & lightweight **BeautifulSoup** parsing.
- 📂 Save data as **CSV, JSON, or text**.

## 🛠️ What You Need
- **Python** 🐍
- **BeautifulSoup 4** 🍜
- **Requests** 📡
- **Pandas** 📊 (for saving data)

## ⚡ Quick Start
1. **Clone this repo** 🛠️
   ```bash
   git clone https://github.com/ErlanggaNursyahbani/WebScrapper-BeautifulSoup-Wikipedia.git
   cd WebScrapper-BeautifulSoup-Wikipedia
   ```
2. **Install dependencies** 📦
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the scraper** 🎯
   ```bash
   python webscraper.py
   ```
